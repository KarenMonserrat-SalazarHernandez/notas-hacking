# GET aHEAD


## Descripción 
Find the flag being held on this server to get ahead of the competition [http://mercury.picoctf.net:45028/](http://mercury.picoctf.net:45028/)

## Pistas
- Maybe you have more than 2 choices
- Check out tools like Burpsuite to modify your requests and look at the responses


## Solución

- Con el comando curl podemos ver el codigo fuente de la página web
- Con el comando -X GET podemos ver la opción de color rojo de la pantalla
- Con el comando -X  POST podemos ver la opción de color azul de la pantalla
